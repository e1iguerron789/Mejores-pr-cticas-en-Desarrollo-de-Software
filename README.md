# Mejores practicas en Desarrollo de Software

Este repositorio contiene ejercicios refactorizados para aplicar los principios SOLID en Java.

## Principios incluidos

- Single Responsibility Principle (SRP)
- Open Closed Principle (OCP)
- Liskov Substitution Principle (LSP)
- Interface Segregation Principle (ISP)
- Dependency Inversion Principle (DIP)

Cada carpeta incluye el codigo fuente refactorizado y un archivo `README.md` con una reflexion sobre como se aplico el principio y que problemas resolvio.
